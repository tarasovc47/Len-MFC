1) Фрейм с разрешёнными сайтами просто перенесен из оригинального html, с единственной правкой всех ссылок target="_blank"
2) Фрейм с инструкцией АИС кувалдой чуть подрихтован, поэтому смотрится чуть красивее. Добавлены комментарии и скриншоты АИС. 
3) классы практически не используются, всё завязано на id
В именах id лаконичность принесена в жертву в пользу читаемости:
"d_" - обозначает вопросы и пункты, где только 1 вариант ответа, то бишь "далее"
"yn_" - элемент подразумевает, что есть 2 ответа: Да и Нет
"y_" - собственно ответ Да к вопросу "yn_"
"n_" - собственно ответ Нет к вопросу "yn_"
"var_" - несколько вариантов ответа
"img_" - id изображений
"view_" - id кнопок, открывающих изображения
"radio_" - id радио-кнопок
"_ok" - id label'ов, к вопросам, начинающимся на "d_"
и их сочетания
